# SpecDock v0.2.1

SpecDock v0.2.1 is a patch release for the v0.2 SDK generation release.

## Fixed

- Generated PHP SDK `composer.json` now includes strict validation metadata.
- GitHub Actions workflows use Node-24-compatible action majors.

## Included From v0.2

- SDK generation for TypeScript, Python, Go, Java, C#, and PHP.
- Runtime target metadata in generated SDK `README.md` and manifest files.
- Generated SDK smoke checks in GitHub Actions and GitLab CI.
