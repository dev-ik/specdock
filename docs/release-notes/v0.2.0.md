# SpecDock v0.2.0

SpecDock v0.2.0 expands SDK generation from TypeScript-only output to six
language targets while keeping generation local-first and file-based.

## Added

- SDK generation for TypeScript, Python, Go, Java, C#, and PHP.
- Runtime target metadata in generated SDK `README.md` and manifest files.
- Generate UI language selector with per-language runtime hints.
- Generated SDK smoke checks in GitHub Actions and GitLab CI.

## Changed

- SDK generation now uses a shared language-neutral model before rendering.
- Generated file diffs are scoped to the active language output.
- Docker and release documentation now point at `v0.2.0`.

## Security

- Proxy behavior is unchanged and remains disabled for public/demo deployments.
- Generated output paths remain relative and traversal-safe.
- Generated SDK code is emitted as files and is not executed by SpecDock.
