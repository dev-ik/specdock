# SpecDock v0.9.0 Internal Milestone

This is an internal milestone, not a public stable release.

## Added

- Electron Builder packaging scripts for unsigned desktop beta artifacts.
- Bundled desktop main/preload build using esbuild.
- Embedded loopback-only Fastify API for packaged desktop runtime.
- Native SpecDock project folder helpers with separate manifest and spec files.
- Native SDK output directory helper with generated path-plan validation.

## Security

- Desktop packaging disables automatic signing identity discovery for internal
  unsigned builds.
- Desktop backend remains loopback-only with proxy and mock routes disabled by
  default.
- Project folders and SDK output paths require explicit user-selected
  directories and reject unsafe writes.
