# SpecDock v0.1.0

Initial public release tag for SpecDock.

> Superseded by `v0.1.2` for Docker image usage. The `v0.1.2` release is the
> first successful published Docker image release.

## Included

- OpenAPI import
- Endpoint explorer
- Request builder
- Response viewer
- TypeScript SDK generation
- ZIP export for generated files
- Docker deployment support

## Notes

Public deployments should keep backend proxy mode disabled:

```env
PUBLIC_DEMO=true
PROXY_ENABLED=false
```

Use the published Docker image from the fixed release:

```bash
docker pull docker.io/d8vik/specdock:v0.1.2
```

See `docs/release-notes/v0.1.2.md` for the current Docker release notes.
