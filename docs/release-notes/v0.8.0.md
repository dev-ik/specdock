# SpecDock v0.8.0 Internal Milestone

This is an internal milestone, not a public stable release.

## Added

- Electron desktop technical preview under `apps/desktop`.
- Loopback-only Fastify API lifecycle managed by the desktop main process.
- Hardened Electron renderer settings and narrow preload IPC.
- Native open/save project file dialog IPC with shared `.specdock.json`
  validation.

## Security

- Desktop startup forces proxy, mock server, and trust-proxy features off by
  default.
- Renderer navigation is limited to the local API origin and new windows are
  denied.
- Native file IPC does not expose arbitrary renderer read/write paths.
