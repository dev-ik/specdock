# SpecDock v1.0.0

SpecDock v1.0.0 is the first public stable desktop-first release.

## Added

- Stable Electron desktop workspace with bundled web UI and embedded Fastify
  backend.
- Native project import/export and project folder workflows.
- SDK generation to local folders through explicit native directory selection.
- Desktop packaging scripts and release checksum generation.
- Portable `.specdock.json` workspace exports with safe request preferences.

## Self-Hosted Web

- Docker Hub image tag: `docker.io/d8vik/specdock:v1.0.0`.
- Public demo and self-hosted proxy safeguards remain unchanged.
- Docker deployments should use immutable version tags and avoid relying on
  `latest`.

## Security

- Desktop installers for this open-source release are unsigned. Verify
  downloaded files with the published `SHA256SUMS.txt` checksum file.
- Desktop renderer runs with Node integration disabled, context isolation, and
  sandboxing.
- Desktop backend binds to loopback only and forces proxy/mock features off by
  default.
- Native filesystem writes are limited to user-selected paths and validated
  project or generated SDK outputs.
- Workspace exports and project folders exclude auth secrets, manual request
  bodies, response bodies, and request history by default.
