# SpecDock v0.2.3

SpecDock v0.2.3 is a public-readiness patch release.

## Fixed

- Includes the public-demo Direct Browser Mode host restriction in the release target.
- Updates Docker examples to use `docker.io/d8vik/specdock:v0.2.3`.
- Aligns storage and security documentation with auth profiles now available in the UI.

## Security

- Public demo deployments keep backend proxy mode disabled.
- When `PUBLIC_DEMO=true`, Direct Browser Mode is limited to `DEMO_DIRECT_ALLOWED_HOSTS`.
- Auth profiles are documented as local browser `localStorage` data that can contain credentials.
- Manual request headers and bodies remain session-only by default.
